# Input demo data for TWAS analysis

The GTEx v10 genotype and small RNA-seq data used for sRNA-TWAS model construction in this study are controlled-access data and are not included in this repository.

This folder describes the expected input format for TWAS model construction and association testing. Users with authorized GTEx v10 access can prepare genotype, small RNA expression, covariate and GWAS summary statistics files following the FUSION input format.

For testing purposes, users can refer to the example data and tutorials provided by FUSION:

- [FUSION](http://gusevlab.org/projects/fusion/) provides TWAS example data and tutorials.

The FUSION demo data can be used to test software installation, input formatting and example TWAS workflows.
